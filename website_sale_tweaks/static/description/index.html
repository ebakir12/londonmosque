Address Management

To configure address fields follow these steps:

  • Go to the Website - Configuration - Websites menu.
  • Choose the necessary website.
  • In the section Billing Address Fields configure the billing address fields.
  • In the section Shipping Address Fields configure the shipping address fields.

To configure the default country:

  • In the section Billing Address Fields select the country that will be used as the default value.
  • Save the changes.

Changing Shop Grid

To set up a product grid in the eCommerce shop:

  • Go to the Website - Configuration - Websites menu.
  • Choose the necessary website.
  • In the section Shop Grid set up a number of products on an one page.
  • Set up a number of columns in the grid.
  • Save the changes.

NOTE: The numbers must have a positive value (greater than 0). Default values: 20 products per page in 4 columns.
