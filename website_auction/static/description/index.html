Odoo Website Auction

Odoo auction module allows admin to add auction on any product. Buyer can bid the product and highest bidder can have purchased the product on the bid amount. On website user can view auction detail prior the auction start, they can also see the time after which the auction is set to active for bidding.

Features:

  • Admin can add auction on active products.
  • Admin can stop the bidding process on the product.
  • Admin can enable/disable the bidder name to be shown on the front end.
  • Admin can enable/disable the winner name to be shown on the front end.
  • Admin can set incremental bidding rule for auction.
  • Admin can set bid records limit to display on website .
  • Customer can set automatic bidding for the product.
  • Customer can subscribe the bids notification .
  • Customer can view the bids details .
  • Buyer can use buy now options if it enabled from backend.
  • Admin can set the status of auction as running, completed or stopped.
  • Different types of bidding supported.
  • Admin can set the notification interval for get notify to extend the bids.
  • Admin can customize the error message in case of any unexpexted result .

Auction Before Start

On website user can view auction detail prior the auction start, they can also see the time after which the auction is set to active for bidding.

There is also feature of subscription for getting update about auction state and bids.There can also buy now option at a price set at backend .

Auction While Running

Once the auction is set to run, website user can start bidding for product. There are two way of bidding simple bid and auto bid .

Auction Extended

The admin can also extend the auction for getting more and more bids .

Auction Details

Auction details can be view by clicking details link.

Here user can view his own current bids(simple as well auto bid) status.

Place Simple Bid

The user can use the simple/non recurring bids for one time bidding .

Place Auto Bid

The Auto Bid option save customer time by setting his max reserve bids for future .The auto bids amount is not displayed to other bidders.

Auctions Information

Here Admin can view the list of auction running and there current bids.

Auction Configuration

Admin configuration auction as per there requirement .He can set the auction time interval , extend date ,add bid increment rule, buy now option here.

Auction Notification

Admin can also configure the notification auction and time threshold for the notification

Auction Publish Information

Admin can configure the auction information to be dispaly on website from here, he can enable/disble the message as per preference .
